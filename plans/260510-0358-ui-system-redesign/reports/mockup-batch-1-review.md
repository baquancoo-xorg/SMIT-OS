## Review checklist (please verify against screenshots)

### Visual identity — LOCK design language batch 1

- [ ] **Glass card pattern**: bg-white/50 backdrop-blur — consistent across cards?
- [ ] **rounded-3xl** (24px) on cards (NOT rounded-2xl 20px)
- [ ] **Page header** với breadcrumb + h2 36px Manrope + italic primary cho second word — đồng bộ giữa Profile và Settings?
- [ ] **Bento metric cards** trên Settings có decorative blob top-right primary/5? (signature element)
- [ ] **Sidebar** active state: bg-primary/10 + border-l-4 primary
- [ ] **Department badges**: Tech blue, Marketing orange, Sale green, Media pink, BOD violet (#6e47ff)

### Color tokens

- [ ] Primary `#0059b6` đúng tone (KHÔNG bị shift về Stitch default blue)
- [ ] Surface `#f7f5ff` lavender tint, NOT pure white
- [ ] Status colors: success green, warning amber, error red — used semantically?
- [ ] NO slate-* colors
- [ ] NO pure black text — all on-surface `#222d51`

### Typography

- [ ] Manrope cho headlines + numerals (tabular-nums trong tables)
- [ ] Inter cho body
- [ ] Form labels uppercase + tracking-widest
- [ ] Italic primary trên word thứ 2 của page title

### Form/UX patterns

- [ ] Input rounded-2xl (16px) với bg-surface-container-low
- [ ] Buttons rounded-full (pill)
- [ ] Touch targets ≥ 44px on mobile
- [ ] Error banner pattern (NOT alert)
- [ ] Verified email badge với check icon
- [ ] 2FA: 6 separate input boxes, monospace, primary focus ring
- [ ] Login error: input border-error/40 + cleared password + lockout warning

### Accessibility

- [ ] Material Symbols icons (NOT emoji)
- [ ] Status colors paired với icons (NOT color-only)
- [ ] Focus visible khi tab key
