# Design Tokens Spec — SMIT-OS

> Status: **Source of truth** as of 2026-05-10 | Phase 2 deliverable
> Implementation: `src/index.css` `@theme` block (Tailwind v4)
> Companion: [`design-system-foundation.md`](./design-system-foundation.md)
> Supersedes: `docs/ui-style-guide.md` (deprecate end of Phase 8)

## Concept — 3-layer architecture

```
PRIMITIVE  (raw values)        SEMANTIC  (meaning)         COMPONENT  (usage)
  #0059b6              →           --color-primary       →    --button-primary-bg
  16px                 →           --space-md            →    --card-padding
  600ms                →           --duration-slower     →    --modal-enter-duration
```

Phase 2 ships **primitive + semantic**. Component tokens emerge in Phase 4.

## How to use

Code references tokens via Tailwind utilities (auto-generated by Tailwind v4 from `@theme`):

```tsx
<div className="bg-primary text-on-primary rounded-3xl shadow-md">    // ✅ tokens
<div className="bg-[#0059b6] text-white rounded-2xl shadow-md">        // ❌ hardcoded
```

For custom CSS:

```css
.my-card {
  padding: var(--space-lg);
  border-radius: var(--radius-card);
  box-shadow: var(--shadow-md);
}
```

| Token | Size | px | Use |
|---|---|---|---|
| `--text-caption` | `0.6875rem` | 11 | Captions, sub-text |
| `--text-label` | `0.75rem` | 12 | Form labels, badges |
| `--text-body-sm` | `0.8125rem` | 13 | Compact body |
| `--text-body` | `0.875rem` | 14 | Default body |
| `--text-body-lg` | `1rem` | 16 | Lead paragraph |
| `--text-h6` | `1.125rem` | 18 | Section heading |
| `--text-h5` | `1.25rem` | 20 | Sub-heading |
| `--text-h4` | `1.5rem` | 24 | Card title |
| `--text-h3` | `1.875rem` | 30 | Section title |
| `--text-h2` | `2.25rem` | 36 | **Page title** |
| `--text-h1` | `3rem` | 48 | Hero |
| `--text-display` | `3.75rem` | 60 | Marketing display |

**Line-height tokens:** `--leading-tight (1.2)`, `--leading-snug (1.35)`, `--leading-base (1.5)`, `--leading-loose (1.75)`.

**Tracking tokens:** `--tracking-tight (-0.02em)`, `--tracking-normal (0)`, `--tracking-wide (0.02em)`, `--tracking-widest (0.1em)`.

**Fonts:**
- `--font-sans: Inter` — body
- `--font-headline: Manrope` — h1-h6 + numerals (data dashboards)

**Note:** `html` font-size scales 13→17px across breakpoints (mobile→2K). All `rem` sizes scale automatically. Use `rem` units, not `px` for typography.

## 3. Spacing scale (clamp-based)

| Token | Min | Fluid | Max | px range |
|---|---|---|---|---|
| `--space-xs` | 0.25rem | 0.5vw | 0.5rem | 4–8 |
| `--space-sm` | 0.5rem | 1vw | 0.75rem | 8–12 |
| `--space-md` | 0.75rem | 1.5vw | 1rem | 12–16 |
| `--space-lg` | 1rem | 2vw | 1.5rem | 16–24 |
| `--space-xl` | 1.5rem | 3vw | 2rem | 24–32 |
| `--space-2xl` | 2rem | 4vw | 3rem | 32–48 |
| `--space-3xl` | 3rem | 6vw | 4rem | 48–64 |

**Why clamp:** spacing scales fluidly with viewport. Mobile gets tight, desktop gets airy, no JS needed.

**Use cases:**

| Token | Pattern |
|---|---|
| `xs` | gap between sibling chips/icons |
| `sm` | gap inside a card row |
| `md` | card inner padding sm screens |
| `lg` | card inner padding lg screens, gap between cards |
| `xl` | section margin, page top padding |
| `2xl` | major section break |
| `3xl` | hero blocks |

## 6. Motion tokens

### Duration

| Token | Value | Use |
|---|---|---|
| `--duration-instant` | 75ms | Hover state changes |
| `--duration-fast` | 150ms | Button press, color shift |
| `--duration-medium` | 250ms | Card expand, tab switch |
| `--duration-slow` | 400ms | Modal enter/exit |
| `--duration-slower` | 600ms | Page transition |

### Easing

| Token | Curve | Use |
|---|---|---|
| `--ease-standard` | `cubic-bezier(0.2, 0, 0, 1)` | Default — Material standard |
| `--ease-decelerate` | `cubic-bezier(0, 0, 0, 1)` | Element entering |
| `--ease-accelerate` | `cubic-bezier(0.3, 0, 1, 1)` | Element leaving |
| `--ease-emphasized` | `cubic-bezier(0.2, 0, 0, 1)` | Signature/hero motion |

**A11y:** `prefers-reduced-motion: reduce` automatically downgrades all animations to 0.01ms (handled in `@layer base`).

**Utilities:** `.motion-fast`, `.motion-medium`, `.motion-slow`, `.ease-standard`.

## 7. Breakpoints (Tailwind defaults)

| Token | Min-width | Tailwind prefix |
|---|---|---|
| `--breakpoint-sm` | 640px | `sm:` |
| `--breakpoint-md` | 768px | `md:` |
| `--breakpoint-lg` | 1024px | `lg:` |
| `--breakpoint-xl` | 1280px | `xl:` |
| `--breakpoint-2xl` | 1536px | `2xl:` |

**Breaking change Phase 2:** old custom breakpoints (xs/tablet/3xl/4xl + reversed xl/2xl) DROPPED. Code with `md:` previously triggered at 430px (custom), NOW triggers at 768px (default). Phase 8 visual smoke test will catch regressions.

**Mental model:**
- mobile-first default (no prefix) = phone portrait
- `sm:` = phone landscape / phablet
- `md:` = tablet portrait
- `lg:` = tablet landscape / small laptop
- `xl:` = desktop
- `2xl:` = wide desktop
