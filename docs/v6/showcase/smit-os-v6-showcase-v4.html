Outline buttons — chỉ border màu primary, icon đổi màu khi hover

Background tối/trong suốt giúp tránh cảm giác "thô". Hover để xem glow + icon transition.

V1 — Pure outline (minimal)

Border mảnh 0.5px cam mờ. Background trong suốt. Hover: border đậm hơn + glow nhẹ phía sau + icon chuyển cam + scale 1.08x. Cảm giác tinh tế nhất, suit cho secondary CTAs hoặc UI có nhiều buttons.

V2 — Tinted background

Background cam mờ 10% + border cam. Hover: background đậm lên 18% + glow mạnh + icon cam sáng + xoay nhẹ -3°. Trung dung — có "presence" nhưng vẫn elegant.

V3 — Animated gradient border (premium)

Wrap gradient cam (xám khi rest, cam sáng khi hover) → tạo hairline cam có sheen ánh sáng. Inner sheen sweep chạy ngang. Premium nhất, dùng cho 1 primary CTA hero per màn.
