Button — 3 levels of polish

Hover từng button để cảm nhận. V3 cao cấp nhất nhờ animated sheen.

V1 — Showcase trước (basic)

Solid fill + halo blur. Phẳng, thiếu chi tiết. Đây là cái bạn thấy "hơi thô".

V2 — Multi-layer (Linear-grade)

Gradient base + glass meniscus + hairline border + shadow tinh tế. Halo nhỏ và ấm. Đã thấy depth thật.

V3 — Animated sheen (Stripe/Vercel)

V2 + sheen sweep chạy ngang khi hover (như ánh sáng quét qua kim loại). Halo lớn hơn. Premium nhất.
